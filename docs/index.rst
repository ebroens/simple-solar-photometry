Simple Solar Photometry Documentation
=====================================

This documentation provides a guide to using the **Simple Solar Photometry** tool,
a tool for extracting 'photometric' data from solar (eclips) images. It generates a data file
which can be used as input for the data file which can be used as input for the 
`Solar Eclipse Models tool`_.
The source code is available on `github`_.
